# Welcome to AzuraJS ‚ö°

AzuraJS is a modern, high-performance web framework designed for **Node.js** and **Bun** environments. It leverages the power of **TypeScript decorators** to provide a clean, intuitive, and robust development experience for building scalable backend applications.

<Callout type="info" title="Ideal for Modern API Development">
  AzuraJS is built for developers who demand speed, type safety, and simplicity
  without the overhead of complex, heavyweight frameworks.
</Callout>

## ‚ú® Why Choose AzuraJS?

| Feature                   | Description                                                                                                         |
| :------------------------ | :------------------------------------------------------------------------------------------------------------------ |
| **TypeScript Native**     | Built with TypeScript from the ground up, offering seamless type safety and superior IntelliSense support.          |
| **Decorator-Driven**      | Use modern decorators to define routes, middleware, and validation, resulting in clean and readable code structure. |
| **High Performance**      | Optimized for speed with minimal overhead and zero unnecessary external dependencies.                               |
| **Lightweight**           | A minimal core with only essential dependencies, preventing `node_modules` bloat.                                   |

| **Flexible Config**       | Support for configuration via TypeScript, JSON, or YAML to suit your workflow preferences.                          |

## üé® Key Features

AzuraJS provides a comprehensive suite of tools for professional API development:

- **Smart Parameter Injection**: Utilize decorators like `@Body`, `@Query`, `@Param`, `@Req`, `@Res` for explicit data access.
- **Built-in Validation**: strong DTO and schema validation system with full type inference.
- **Cookie Management**: Native support for safe cookie parsing and serialization.
- **Production-Ready CORS**: Configurable CORS plugin designed for security and flexibility.
- **Rate Limiting**: Integrated protection against abuse and DDoS attacks with per-route configuration.
- **Cluster & Proxy**: Native cluster support for multi-core performance and trusted proxy management.
- **Contextual Logging**: Adaptive logging system for effective monitoring and debugging.
- **Structured Error Handling**: Centralized error management with `HttpError` and dedicated middleware.

## üöÄ Simple & Powerful

Create a fully functional CRUD API with minimal boilerplate. The following example demonstrates a complete controller implementation:

```typescript
import { AzuraClient } from "azurajs";
import {
  applyDecorators,
  Controller,
  Get,
  Post,
  Body,
  Param,
} from "azurajs/decorators";

@Controller("/api/users")
class UserController {
  @Get()
  getAllUsers() {
    return { users: ["Alice", "Bob", "Charlie"] };
  }

  @Get("/:id")
  getUser(@Param("id") id: string) {
    return { id, name: `User ${id}` };
  }

  @Post()
  createUser(@Body() data: unknown) {
    return { id: Date.now(), ...data };
  }
}

const app = new AzuraClient();
applyDecorators(app, [UserController]);
await app.listen();
```
