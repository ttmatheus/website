  <Tab value="bun">```bash bun add azurajs ```</Tab>

## TypeScript Configuration ğŸ› ï¸

AzuraJS relies on **decorators** to provide its expressive syntax. You must enable them in your `tsconfig.json` file.

Add or update the following compiler options:

```json
{
  "compilerOptions": {
    "target": "ESNext",
    "module": "ESNext",
    "experimentalDecorators": true,
    "emitDecoratorMetadata": true,
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true,
    "forceConsistentCasingInFileNames": true
  }
}
```
