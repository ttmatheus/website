# Rate Limiting üõ°Ô∏è

AzuraJS includes a built-in Rate Limiting middleware designed to safeguard your API against brute-force attacks, abuse, and Denial-of-Service (DDoS) attempts by restricting the number of requests a client can execute within a specific timeframe.

## Global Configuration

Configure global Rate Limiting settings within your `azura.config.ts`.

```typescript
// azura.config.ts
import type { ConfigTypes } from "azurajs/config";

const config: ConfigTypes = {
  plugins: {
    rateLimit: {
      enabled: true,
      limit: 100,
      timeframe: 60000, // 1 minute
      message: "Too many requests, please try again later.",
      trustProxy: false,
    },
  },
};

export default config;
```

### `rateLimit` Options

| Property     | Type      | Description                                                                   |
| :----------- | :-------- | :---------------------------------------------------------------------------- |
| `enabled`    | `boolean` | Activates the Rate Limiting middleware. Default: `false`.                     |
| `limit`      | `number`  | Maximum allowed requests within the timeframe. Default: `60`.                 |
| `timeframe`  | `number`  | Window duration in milliseconds. Default: `60000` (1 minute).                 |
| `message`    | `string`  | Custom error response message.                                                |
| `trustProxy` | `boolean` | Resolves client IP via `X-Forwarded-For` if behind a proxy. Default: `false`. |

## Per Route/Controller Rate Limiting

Apply Rate Limiting selectively to specific controllers or routes using the `@UseMiddleware` decorator in conjunction with the `rateLimit` utility.

```typescript
import { Controller, Get, Post, UseMiddleware } from "azurajs/decorators";
import { rateLimit } from "azurajs/plugins";

const tenRequestsPerMinute = rateLimit(10, 60 * 1000);

@Controller("/auth")
export class AuthController {
  @UseMiddleware(tenRequestsPerMinute)
  @Get("/login")
  login() {
    return "Login Page";
  }

  @UseMiddleware(rateLimit(5, 30 * 1000))
  @Post("/register")
  register() {
    return "User Registration";
  }
}
```

## How It Works

The middleware tracks request counts per unique IP (or identifier if `trustProxy` is enabled) during the specified window. Exceeding requests trigger a `429 Too Many Requests` response.
