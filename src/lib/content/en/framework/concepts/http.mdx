# Request & Response ðŸ’¬

In AzuraJS, the request (`RequestServer`) and response (`ResponseServer`) objects are the primary interface for interacting with the HTTP client. They are based on the standard Node.js specification but extended with additional functionalities to facilitate API development.

## `RequestServer` Object

The `RequestServer` object represents the HTTP request received from the client. It contains information about the request, such as headers, URL parameters, request body, and much more.

### Common Properties

| Property | Type | Description |
| :--- | :--- | :--- |
| `url` | `string` | The full URL of the request. |
| `method` | `string` | The HTTP method of the request (GET, POST, etc.). |
| `headers` | `Record<string, string>` | An object containing all request headers. |
| `params` | `Record<string, string>` | Route parameters extracted from the URL (e.g., `/users/:id`). |
| `query` | `Record<string, string>` | Query string parameters from the URL (e.g., `?name=value`). |
| `body` | `any` | The request body, already parsed (JSON, form, etc.). |
| `ip` | `string` | The IP address of the client that made the request. |
| `cookies` | `Record<string, string>` | An object containing all request cookies. |

### Common Methods

| Method | Description |
| :--- | :--- |
| `json(): Promise<any>` | Reads and parses the request body as JSON. |
| `text(): Promise<string>` | Reads the request body as text. |
| `arrayBuffer(): Promise<ArrayBuffer>` | Reads the request body as an ArrayBuffer. |
| `blob(): Promise<Blob>` | Reads the request body as a Blob. |
| `formData(): Promise<FormData>` | Reads the request body as FormData. |

### Usage Example

```typescript
import { Controller, Get, Post, Req, Body, Param, Query } from "azurajs/decorators";
import type { RequestServer } from "azurajs/types";

@Controller("/data")
export class DataController {
  @Get("/:id")
  getData(
    @Req() req: RequestServer,
    @Param("id") id: string,
    @Query("filter") filter?: string
  ) {
    console.log("Method:", req.method);
    console.log("URL:", req.url);
    console.log("Headers:", req.headers);
    return { id, filter, ip: req.ip };
  }

  @Post()
  async postData(@Body() body: any) {
    // The @Body decorator already parses, but you can use req.json() in pure JS
    return { received: body };
  }
}
```

## `ResponseServer` Object

The `ResponseServer` object represents the HTTP response that will be sent back to the client. It allows you to define the status, headers, and body of the response.

### Common Properties

| Property | Type | Description |
| :--- | :--- | :--- |
| `statusCode` | `number` | The HTTP status code of the response. |
| `headersSent` | `boolean` | Indicates whether headers have already been sent. |

### Common Methods

| Method | Description |
| :--- | :--- |
| `status(code: number): this` | Sets the HTTP status code of the response. |
| `json(data: any): void` | Sends a JSON response. |
| `send(body: string | Buffer | object): void` | Sends a text, buffer, or JSON response. |
| `end(): void` | Ends the response without sending data. |
| `setHeader(name: string, value: string): void` | Sets a response header. |
| `getHeader(name: string): string | undefined` | Gets the value of a response header. |
| `cookie(name: string, value: string, options?: CookieOptions): void` | Sets a cookie. |
| `clearCookie(name: string, options?: CookieOptions): void` | Deletes a cookie. |
| `redirect(url: string, status?: number): void` | Redirects the client to another URL. |

### Usage Example

```typescript
import { Controller, Get, Res } from "azurajs/decorators";
import type { ResponseServer } from "azurajs/types";

@Controller("/status")
export class StatusController {
  @Get("/ok")
  ok(@Res() res: ResponseServer) {
    res.status(200).send("All good!");
  }

  @Get("/error")
  error(@Res() res: ResponseServer) {
    res.status(500).json({ message: "Internal server error" });
  }

  @Get("/redirect")
  redirectToGoogle(@Res() res: ResponseServer) {
    res.redirect("https://www.google.com");
  }
}
```
