# Request & Response ðŸ’¬

In AzuraJS, the `RequestServer` and `ResponseServer` objects act as the primary interface for HTTP interactions. While based on standard Node.js specifications, they are enhanced with convenient methods and properties to streamline API development.

## `RequestServer` Object

The `RequestServer` object encapsulates the incoming HTTP request, providing access to headers, URL parameters, the request body, and client information.

### Common Properties

| Property  | Type                     | Description                                      |
| :-------- | :----------------------- | :----------------------------------------------- |
| `url`     | `string`                 | The full request URL.                            |
| `method`  | `string`                 | The HTTP method (GET, POST, etc.).               |
| `headers` | `Record<string, string>` | Collection of request headers.                   |
| `params`  | `Record<string, string>` | Route parameters extracted from the URL path.    |
| `query`   | `Record<string, string>` | Parsed query string parameters.                  |
| `body`    | `any`                    | The parsed request body (JSON, form data, etc.). |
| `ip`      | `string`                 | The client's IP address.                         |
| `cookies` | `Record<string, string>` | Collection of request cookies.                   |

### Common Methods

| Method                                | Description                             |
| :------------------------------------ | :-------------------------------------- |
| `json(): Promise<any>`                | Asynchronously parses the body as JSON. |
| `text(): Promise<string>`             | Asynchronously reads the body as text.  |
| `arrayBuffer(): Promise<ArrayBuffer>` | Reads the body as an ArrayBuffer.       |
| `blob(): Promise<Blob>`               | Reads the body as a Blob.               |
| `formData(): Promise<FormData>`       | Reads the body as FormData.             |

### Usage Example

```typescript
import {
  Controller,
  Get,
  Post,
  Req,
  Body,
  Param,
  Query,
} from "azurajs/decorators";
import type { RequestServer } from "azurajs/types";

@Controller("/data")
export class DataController {
  @Get("/:id")
  getData(
    @Req() req: RequestServer,
    @Param("id") id: string,
    @Query("filter") filter?: string,
  ) {
    return {
      id,
      filter,
      ip: req.ip,
      method: req.method,
      url: req.url,
    };
  }

  @Post()
  async postData(@Body() body: any) {
    return { received: body };
  }
}
```

## `ResponseServer` Object

The `ResponseServer` object facilitates sending the HTTP response back to the client, allowing you to set status codes, headers, cookies, and the response body.

### Common Properties

| Property      | Type      | Description                                   |
| :------------ | :-------- | :-------------------------------------------- |
| `statusCode`  | `number`  | The current HTTP status code.                 |
| `headersSent` | `boolean` | Boolean indicating if headers have been sent. |

### Common Methods

| Method                                                               | Description                             |
| :------------------------------------------------------------------- | :-------------------------------------- | -------------------------------- | ----------------------------------------------- |
| `status(code: number): this`                                         | Sets the HTTP status code.              |
| `json(data: any): void`                                              | Sends a JSON response.                  |
| `send(body: string                                                   | Buffer                                  | object): void`                   | Sends a response of type text, buffer, or JSON. |
| `end(): void`                                                        | Ends the response without sending data. |
| `setHeader(name: string, value: string): void`                       | Sets a specific response header.        |
| `getHeader(name: string): string                                     | undefined`                              | Retrieves a set response header. |
| `cookie(name: string, value: string, options?: CookieOptions): void` | Sets a cookie with options.             |
| `clearCookie(name: string, options?: CookieOptions): void`           | Clears a specific cookie.               |
| `redirect(url: string, status?: number): void`                       | Redirects the request to a new URL.     |

### Usage Example

```typescript
import { Controller, Get, Res } from "azurajs/decorators";
import type { ResponseServer } from "azurajs/types";

@Controller("/status")
export class StatusController {
  @Get("/ok")
  ok(@Res() res: ResponseServer) {
    res.status(200).send("All good!");
  }

  @Get("/error")
  error(@Res() res: ResponseServer) {
    res.status(500).json({ message: "Internal server error" });
  }

  @Get("/redirect")
  redirectToGoogle(@Res() res: ResponseServer) {
    res.redirect("https://www.google.com");
  }
}
```
